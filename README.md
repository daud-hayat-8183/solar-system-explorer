# 🌌 Cosmos Explorers — Solar System Explorer

An **interactive educational web application** for children and students to explore our Solar System! Built with React, TypeScript, Vite, and TailwindCSS.

## 🚀 Live Demo

**[Visit the Website →](https://daud-hayat-8183.github.io/solar-system-explorer/)**

## ✨ Features

- **Interactive Solar System Canvas** — Watch planets orbit the Sun in real-time with adjustable speed and zoom controls
- **Planet Explorer** — Browse all celestial bodies with detailed spectral analysis, core composition breakdowns, and atmospheric data
- **Comparison Lab** — Compare planets side-by-side with visual scale representations
- **Quiz System** — Test your space knowledge with multiple-choice quizzes and earn XP badges
- **Virtual Mission Simulator** — Pilot telemetry probes to retrieve data from planets
- **Sound Effects** — Browser-synthesized audio feedback for quiz interactions
- **Progress Tracking** — XP points, achievement badges, and mission counts saved locally

## 🛠️ Tech Stack

- **React 19** + **TypeScript**
- **Vite** — Fast build tool
- **TailwindCSS 4** — Utility-first CSS
- **Framer Motion** — Smooth animations
- **Lucide React** — Icon library
- **HTML5 Canvas** — Solar system orbital simulation

## 📦 Getting Started (Local Development)

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## 🌐 Deployment

This project automatically deploys to GitHub Pages via GitHub Actions on every push to the `main` branch.
